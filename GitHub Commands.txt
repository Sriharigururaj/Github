GitHub Commands from basic to advanced for MAC:

git version - to get the current version of GitHub if installed on your device.

git config —global user.email “ [Sriharigururaj@gmail.com]

git config —global core.editor “/Applications/Sublime\ Text.app/Contents/MacOS/Sublime\

p4merge/“

git config —global mergetool.prompt false
